# bombdsys
数据轰炸，测试于微信QQ均可测试（测试交流用不要乱使用

## 发送内置数据库

*按顺序发送数据*
`run god i`

*随机发送数据默认发20条*
`run god r`

*你也可以加上发送次数*
`run god r 50`
*发送50条*

## 输入help可获取帮助
